Invoice
=======

Facture HTML & CSS éditable/imprimable pour auto-entrepreneur.

[Démo](http://dflorent.github.io/invoice/)

Pré-requis
==========

- ndenv (https://github.com/riywo/ndenv)

Avant de commencer
==================

```
ndevn install
npm install
```

Lancer l'application
====================

```
npm start
```
